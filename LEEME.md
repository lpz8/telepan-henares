# TELEPAN HENARES — App de Gestión

## CÓMO USAR

1. Abre la carpeta `frontend` en VS Code
2. Abre el terminal (Ctrl+Ñ) y ejecuta:
   ```
   npm install
   npm run dev
   ```
3. Abre el navegador en http://localhost:3000
4. Login: telepansgl@gmail.com / tu contraseña

## PARA PUBLICAR EN VERCEL

1. Sube el proyecto a GitHub
2. Ve a vercel.com → New Project → selecciona el repo
3. Root Directory: `frontend`
4. Variables de entorno (ya están en el código, no hace falta)
5. Deploy
