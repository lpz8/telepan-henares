OPCIÓN A — Local (en tu PC):
  1. Abre el archivo "frontend/.env"
  2. Cambia PON_AQUI_TU_ANON_KEY por tu anon key real
  3. Guarda el archivo
  4. Abre terminal en carpeta "frontend"
  5. Escribe: npm install --legacy-peer-deps
  6. Escribe: npm run dev
  7. Abre: http://localhost:3000
  8. Email: telepansgl@gmail.com
  9. Contraseña: la que tienes en Supabase → Authentication

OPCIÓN B — Vercel (para móvil y cualquier lugar, GRATIS):
  1. Sube la carpeta "frontend" a GitHub como repositorio
  2. Ve a https://vercel.com → New Project → importa el repo
  3. Root Directory: frontend (o deja vacío si solo subiste frontend)
  4. Build Command: npm install --legacy-peer-deps && npm run build
  5. Output Directory: dist
  6. Environment Variables → añade:
       VITE_SUPABASE_URL  = https://cbozdjosionujkrxhwz.supabase.co
       VITE_SUPABASE_ANON_KEY = tu_anon_key
  7. Deploy → en 2 minutos tienes la URL
  8. Instala en iPhone: Safari → Compartir → "Añadir a inicio"
  9. Instala en Android: Chrome → Menú → "Instalar aplicación"

OPCIÓN C — Netlify (alternativa a Vercel):
